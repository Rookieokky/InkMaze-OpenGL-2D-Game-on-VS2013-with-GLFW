編譯環境：Visual Studio 2013
作業系統：Win7 x64

遊戲目的：
	在橫向卷軸的世界中找到唯一的出口

操作說明：
	鍵盤左鍵：角色向左移動
	鍵盤右鍵：角色向右移動
	空白建或X：跳躍
	(在牆壁上按方向鍵以抓住牆壁，放開方向鍵後按跳躍鍵可進行蹬牆跳)
	ESC：關閉遊戲
	R：重新開始
	F12：開起/關閉 開發人員模式(為遊戲品質不建議開啟)

特別說明：
	本遊戲使用GLFW為主要架構所寫成
	另為此專案所使用的的外部函式庫包括：
		GLFW, GLEW,FreeImage, Freetype2,  FTGL, Box2D

	FreeImage用於改寫原本RGBpixmap讀入圖片的功能，原先的readBMPfile函式被我刪除，並增加名為 loadTexture 的函式，藉著FreeImage的功能此函式得以載入的圖片格式多達20種以上。

	Freetype2 與 FTGL 用於讓文字可以顯示在畫面上，因為glfw原先內建沒有可以顯示文字的方式，另外靠著這兩個函式庫我可以使用所有ttf格式的字型檔。

	Box2D為 2D 遊戲的物理引擎，在遊戲中角色所有的移動與碰撞的數據運算接以此物裡引擎得以實現，如果想看實際上物件在物理引擎中的建構型態請按F12
